# 论文库

这里保存安全四大会论文笔记。每篇论文建议使用 `content/templates/论文笔记模板.md`。

## 入口

- [[论文收集箱]]
- [[论文获取与元数据]]
- [[venues/index|安全四大会]]

## 推荐命名

```text
content/papers/YYYY/VENUE/short-title.md
```

示例：

```text
content/papers/2026/CCS/example-paper-title.md
```

## 状态标签

- `status/unread`
- `status/reading`
- `status/summarized`
- `status/reproducing`
- `status/reproduced`
- `status/blocked`

## 每篇论文至少回答

1. 这篇论文解决什么问题？
2. 它和已有工作相比新在哪里？
3. 实验是否支撑结论？
4. 能不能复现，复现成本在哪里？
5. 这条线下一篇该读什么？
