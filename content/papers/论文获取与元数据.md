# 论文获取与元数据

## 推荐优先级

1. 会议官网或出版社页面。
2. DBLP 获取题名、作者、年份、DOI、出版社链接。
3. 作者主页、项目主页、机构仓库。
4. arXiv、IACR ePrint 或其他预印本平台。
5. Zotero 管理 PDF、BibTeX 和阅读状态。

## 四大会入口

- [[CCS]]
- [[IEEE S&P]]
- [[USENIX Security]]

## 元数据字段

每篇论文笔记建议保留：

```yaml
title:
venue:
year:
authors:
doi:
dblp:
official:
pdf:
code:
artifact:
dataset:
status:
tags:
```

## 下载原则

- 优先使用官方公开 PDF、作者开放版本或机构仓库版本。
- 出版社页面只提供摘要时，不要把受限 PDF 直接上传到公开仓库。
- 若需要保存 PDF，建议放到 Zotero 或本地私有目录，不要默认发布到 Quartz 网站。
- 对每篇论文记录“PDF 来源”和“是否为正式版本/预印本”。

## 自动化计划

- 第一步：从 DBLP 拉取年度论文清单。
- 第二步：为每篇论文生成 Markdown stub。
- 第三步：用 Zotero/Better BibTeX 补 citation key 和 BibTeX。
- 第四步：人工确认 PDF、代码、artifact、数据集链接。
