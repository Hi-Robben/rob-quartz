# 论文获取与元数据

## 推荐优先级

1. 会议官网或出版社页面。
2. DBLP 获取题名、作者、年份、DOI、出版社链接。
3. 作者主页、项目主页、机构仓库。
4. arXiv、IACR ePrint 或其他预印本平台。
5. Zotero 管理 PDF、BibTeX 和阅读状态。

## 四大会入口

- [[CCS]]
- [[IEEE S&P]]
- [[USENIX Security]]
- [[NDSS]]

## 元数据字段

每篇论文笔记建议保留：

```yaml
title:
venue:
year:
authors:
doi:
dblp:
official:
pdf:
code:
artifact:
dataset:
status:
tags:
```

## 下载原则

- 优先使用官方公开 PDF、作者开放版本或机构仓库版本。
- 出版社页面只提供摘要时，不要把受限 PDF 直接上传到公开仓库。
- 若需要保存 PDF，建议放到 Zotero 或本地私有目录，不要默认发布到 Quartz 网站。
- 对每篇论文记录“PDF 来源”和“是否为正式版本/预印本”。

## 自动化计划

- 第一步：从 DBLP 拉取年度论文清单。
- 第二步：为每篇论文生成 Markdown stub。
- 第三步：用 Zotero/Better BibTeX 补 citation key 和 BibTeX。
- 第四步：人工确认 PDF、代码、artifact、数据集链接。

## 当前脚本

生成 2025 四大会论文表：

```powershell
node scripts/collect-2025-security-papers.mjs
```

同时解析开放 PDF 链接，但不下载文件：

```powershell
node scripts/collect-2025-security-papers.mjs --resolve-pdfs --concurrency 3
```

下载已找到的开放 PDF 到本地私有目录：

```powershell
node scripts/collect-2025-security-papers.mjs --resolve-pdfs --download-pdfs --concurrency 2
```

PDF 会保存到 `local-library/pdfs/2025/`。该目录已被 `.gitignore` 忽略，不会发布到网站。

生成 Zotero 导入文件：

```powershell
node scripts/export-zotero-ris.mjs
```

导入文件保存在：

```text
local-library/zotero-import/
```

推荐先导入：

```text
2025-security-all.notes-only.ris
```

这个文件会导入 2025 四大会全部 1303 条论文元数据，并把开放 PDF 链接写入 Zotero 备注，不会下载 PDF。若想让 Zotero 尝试把 PDF URL 保存成链接附件，再测试：

```text
2025-security-all.with-pdf-links.ris
```

## PDF 状态含义

- `找到开放 PDF`：脚本找到了明确开放访问的 PDF 链接。
- `已下载`：PDF 已下载到本地 `local-library/`。
- `未找到`：DBLP/官方页/OpenAlex 未发现开放 PDF，需要人工查作者主页、项目页或 Zotero。
- `检查失败`：访问官方页失败，需要稍后重试或手动检查。
