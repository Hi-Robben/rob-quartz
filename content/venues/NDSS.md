# NDSS

Network and Distributed System Security Symposium，安全与隐私领域综合顶会之一。

## 关注重点

- 网络与分布式系统安全
- Web、移动和平台安全
- 隐私、安全协议和测量
- IoT、CPS、云与基础设施安全
- 攻击检测、防御和实证研究
